Brandon Thompson

Hi there! My name is Brandon Thompson, I've recently graduated from Teesside Univeristy with a Bachelor Of Science in Technical Game Development with First Class Honours!
I'm an aspiring game developer and below is my portfolio, please take a look!

Sprint

This game is a project made for the final year project module at Teesside University. The goal of this game is to run to the end of each level as fast as possible, setting and / or beating your own personal times!

Experimental Flight Movement

This project was created to experiment with super-hero-like movement, this demo allows the player to fly, go super-sonic, run on the ground, and sprint. The player is also able to seamlessly transition between flying and not flying.
